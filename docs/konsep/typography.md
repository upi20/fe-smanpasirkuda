# Typography

Acuan dari referensi:
- Body: Lato, weight 400
- Nav & Heading: Lato, weight 700
- Accent/alt: Dancing Script, weight 400

## Font family
- Base: `Lato, sans-serif`
- Heading: `Lato, sans-serif`
- Accent: `Dancing Script, sans-serif`

## Aturan pemakaian
- Heading (H1–H6) pakai Lato bold.
- Navigasi/topbar pakai Lato bold.
- Accent (Dancing Script) hanya untuk elemen dekoratif/aksen (mis. tagline kecil), bukan untuk paragraf panjang.

## Warna teks
- Default teks: `#0a0a0a` pada surface putih/abu.
- Pada area primary/secondary: teks putih.
